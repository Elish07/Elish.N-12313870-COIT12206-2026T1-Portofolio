## Overview
This task introduces the basic use of GNS3 for network simulation. It focuses on creating a simple project, configuring a Linux host, and verifying IP address settings using command-line tools.

## Task Description
A GNS3 project was created with a single Linux host. The host was configured with a static IP address using system configuration files. The setup was tested by running commands in the terminal to confirm the configuration.

## Steps Performed
- Created a new GNS3 project  
- Added a Linux host node  
- Assigned a static IP address  
- Configured network settings using /etc/network/interfaces  
- Started the node and opened the console  
- Verified IP address using command
